Questo progetto prevede l'utilizzo di C per la generazione di un eseguibile che consenta di proporre una codifica di Huffman efficiente data una stringa passata come parametro al momento del lancio dell'eseguibile. 
ATTENZIONE!!! La stringa dovrà essere delimitata da appositi caratteri come apici, doppi apici, ... .
In caso contrario semplicemente il programma restituirà errore perchè riconoscera il parametro come doppio.
